# Meetup AngularJS

Esta aplicación fue desarrollada por el grupo Semillero de Innovación Geográfica (SIG) de Esri Colombia. Para mayor información consulte http://desarrolladores.esri.co/ 
Síguenos en:

-	Twitter: @geo_geeks
-	Facebook: https://www.facebook.com/geogeeksco/ 
-	YouTube:  https://www.youtube.com/channel/UCuGsuNbUykWZ6lsb85PeW0A
-	Meetup: http://www.meetup.com/GeoGeeks/

Aprende cómo crear aplicaciones web con mapas de ArcGIS de forma fácil utilizando el framework de AngularJS para ArcGIS. El propósito de este repositorio es hacer fácil el uso de mapas y servicios de Esri en aplicaciones angular. No te pierdas este MeetUp y averigua cómo.


## Instrucciones 

Descargue el proyecto y descomprímalo en su servido local.
Para más información sobre AngularJS aplicado a la tecnología de ArcGIS consulte: 

-	GitHub esri: https://github.com/esri/angular-esri-map
-	Ejemplos: http://esri.github.io/angular-esri-map/#/home
